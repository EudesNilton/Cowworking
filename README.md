
# Cowworking

Onde estão todas, ou quase todas, as vagas de estágio e processos seletivos abertos.


## Funcionalidades

- Temas dark e light
- Encontrar vagas por localidade, curso ou área de atuação
- Multiplataforma


## Stack utilizada

**Front-end:** NextJs

**Back-end:** C#, ASP.Net, EntityFramework, Swagger

**Banco de Dados:** SQL Server
